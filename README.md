# Crecen-Sanos-App
Proyecto de titulación - App Veterinaria
## Tecnologías y Frameworks
* **Motor:** Unity 6 (6000.4.2f1)
Sistema de Eventos de Unity (No-Code/Visual Interaction).
* **Plataforma:** Android (Optimizado para dispositivos como Redmi Note 15 Pro Plus)
* **UI:** Unity UI + TextMeshPro
* **Control de Versiones:** Git / GitHub
## Instrucciones de Ejecución
1. Clonar el repositorio.
2. Abrir con Unity Hub usando la versión Unity 6.
3. Asegurarse de que el proyecto esté en plataforma **Android** (File > Build Settings).
4. La escena principal se encuentra en `Assets/Scenes/SampleScene`.

# Guía de Uso: Aplicación de Gestión y Salud de Mascotas

Bienvenido a la guía oficial de uso. A continuación, se detallan los pasos y funcionalidades disponibles para navegar correctamente por la aplicación y aprovechar sus recursos de salud y alimentación.

## 1. Inicio y Registro
Al ejecutar la aplicación, se presentará el **Menú Principal**. Para comenzar, siga estas rutas:

* **Botón Comenzar:** Le dirigirá al apartado de **Registro de Mascotas**.
    * Aquí podrá ingresar el nombre de su mascota y seleccionar su especie (Perro o Gato).
    * Al presionar **Continuar**, la aplicación personalizará la experiencia según la especie elegida.
* **Botón Registrar Después:** Si prefiere no ingresar datos de inmediato, esta opción le permitirá acceder directamente a la sección de consulta de información general para perros o gatos.

## 2. Cronograma de Vacunación y Salud
Una vez seleccionada la mascota, el sistema le llevará automáticamente al **Cronograma de Vacunación**:

* **Dosis y Frecuencia:** Se especifican las vacunas necesarias y los tiempos recomendados para cada mascota.
* **Sección de Enfermedades:** En la parte inferior, encontrará el botón **Enfermedades**.
    * Al activarlo, verá un listado de patologías que se previenen con el cronograma mostrado.
    * **Detalles y Sintomatología:** Al seleccionar una enfermedad específica, se desplegará información detallada sobre qué es, sus síntomas y una advertencia de seguridad instando a la consulta veterinaria profesional.

## 3. Alimentación
A través de la barra de navegación inferior, podrá acceder al apartado de **Comida**:

* El contenido se adaptará automáticamente a la especie seleccionada (Perro/Gato).
* **Filtros de Alimentos:** Dentro de esta pestaña, encontrará dos secciones críticas:
    1.  **Apto:** Listado de alimentos seguros para el consumo del animal.
    2.  **No Apto:** Alimentos prohibidos o peligrosos que deben evitarse.

## 4. Navegación Global
Para facilitar el uso, todas las pestañas cuentan con una barra de navegación inferior con los siguientes accesos directos:

* **Vacunas:** Regresa al cronograma de vacunación correspondiente a su mascota actual.
* **Comida:** Dirige a la sección de pautas alimenticias.
* **Home (Casa):** Este apartado se encuentra actualmente **bajo exploración** y será habilitado con nuevas funcionalidades en futuras actualizaciones.

---
*Nota: Esta aplicación es una herramienta informativa. Ante cualquier síntoma persistente en su mascota, acuda de inmediato a un médico veterinario.*
